# MediMart Pharmacy Angular Client - Phase 2 to 5

Custom Angular 20 client inspired by the supplied Propharm pharmacy reference styling. This project does not copy proprietary theme source code.

## Included
- Homepage with pharmacy style header, mega menu, hero, categories, banners, featured products, best sellers and newsletter
- Shop with 1000 generated products, search, category/brand filter, price filter, prescription filter, sorting and pagination
- Category pages and brand pages
- Product detail page with gallery, stock, prescription label, reviews, related products, wishlist, compare, add-to-cart and buy-now
- Wishlist, compare, cart, checkout, payment option selection, orders and order tracking
- User profile and address management
- Admin pages for dashboard, products, categories, brands, inventory, suppliers, orders, payments, customers, reports, coupons and settings
- Dockerfile, nginx.conf, Jenkinsfile, .gitignore and VS Code settings

## Run
npm install
npm start

Open http://localhost:4200
